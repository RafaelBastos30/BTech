Formatação de texto

⬅ Início

Strong(<strong>)

Serve como negrito, tem a função de deixar uma palavra marcada com a coloração mais forte!

Ênfase(<em>)

Serve como itálico, mudando a caligrafía das paralavras!

Marcação(<mark>)

Serve como um marca texto, passando uma faixa colorida e transparente pelas palavras!

Deletar(<del>)

Serve para marcar textos como excluídos, eles ficam marcados com uma linha atravessando-o!

Inserir(<ins>)

Serve para mostrar que um texto foi adicionado, e fica sublinhado!

Subrescrito(<sup>)

Serve para por um texto sobrescrito, por exemplo x12, Olá Mundo

Subscrito(<sub>)

Serve para deixar um texto Subscrito, por exemplo: H2O, Olá Mundo!

Código-fonte(<code>)

Serve para deixar um texto com todos caracteres do mesmo tramanho, por exemplo: Olá, mundo, tudo bem?

PRE(<pre>)

Serve para deixar exposto no texto as caracteriscas do código, como a identação, mas acaba sendo pouco responsivo, por exemplo:

Como diria Madara Uchira: Onde há luz sempre haverá escuridão!
