⬅ Início

Criando a estrutura do nosso arquivo

Para iniciarmos o nosso arquivo html é necessário que criemos algumas tags muito importantes para organizar tudo. Sendo elas as tags a seguir:

<header>

Para indicarmos a área em que serão inseridos os conteúdos do nosso cabeçalho!

<main>

Para indicarmos a área que será o corpo do nosso arquivo, onde serão inseridos os nossos conteúdos!

<footer>

Para indicarmos a área em que serão inseridos os conteúdos do nosso rodapé!

Marcações de texto

Dentro da nossa estrutura nos podemos inserir nossos dados e para isso utilizamos as seguintes tags:

<p>

É o comando utilizado para criar um parágrafo!

<h1>

Para criarmos um título, podendo ir até o h6, sendo o h1 o título principal e o h6 otítulo com a menor hierarquia.

<a>

Para inserirmos links no nosso conteúdo.

Ao digitar a letra "a" no VScode e apertar enter ele automaticamente ira digitar o comando completo, que é <a href=""></a>
