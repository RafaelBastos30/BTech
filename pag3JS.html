⬅ Início

Criando a estrutura do nosso código

Para iniciarmos o nosso arquivo JavaScript é necessário que criemos uma tag munto. Sendo ela a tag a seguir:

<script>

Para indicarmos a área em que serão inseridos os conteúdos do JavaScript!

Criando nossa primeira janela!

Dentro da nossa tag <scricpt> nos podemos inserir nossos comandos, como nos exemplos a seguir:

alert("olá")

É o comando utilizado para criar um alerta na nossa tela, contendo apenas um texto!

prompt("Qual seu nome?")

Para criarmos uma janela semelhante ao alerta, porem né existe uma caixa de texto para inserirmos dados.

confirm("Você é um humano?")

Este comando cria uma janela de comfirmação, com botões de confirmação e negação.
