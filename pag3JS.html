⬅ Início

Declarando variaveis

Para declarar uma variavel em java script é muito simples, a princípio utilizamos o sehuinte comando:

var nome_da_variavel = "Olá mundo!"

var nome_da_variavel = 5

var nome_da_variavel = true

Diferente de outras linguagens, como java, no JavaScript nós não precisamos indicar que tipo de variável estamos declarando, pois o javascript é capaz de identificar automaticamente ao inserirmos o dado.

Mesclando nossos conhecimentos

Se lembra do tema anterior? A criação de janelas? Pois então, agora nós podemos armazenar os dados que são inseridos lá!

Para armazenar esses dádor iremos criar um prompt, só que da seguinte maneira:

var nome = prompt("Fala meu nobre! Qual seu nome?");

Desse modo estarei indicando que a variavel nome terá como atribuição o valor que for digitado pelo usuário!

Indo além!

Já que aprendemos a atribuir dados digitados nas janelas para as variáveis, que tal irmos além

Nesse exemplo iremos somar dois números digitados em prompts:

var n1 = Number(prompt(nome+", preciso que você digite um número:"));

var n2 = Number(prompt(nome+", preciso que você digite outro número:"));

n3 = n1 + n2;

alert("A soma dos números é igual a: "+n3)

Neste caso pusemos o prompt dentro do comando "Number" para transformarmoso dado em um número, já que como foi dito no ensinamento sobre janelas, tudo que é digitado lá se torna altomaticamente uma string!

Criamos a variavel n3 para receber o valor da soma de n1 e n2!

E por fim criamos um alert para nos mostrar o resultado da soma! no fim do texto você pode observar que pusemos o seguinte comando:

+n3

Nesta situação o + não serve para adição, e sim para concatenação, sendo assim, ao abrir e ou ao fechar aspas nós temos que por um + para concatenar com nossa variavel. seja ela string ou number, e exibila!

Dica!

Outra maneira de concatenar nossas variaveis ao exibir um texto, é utilizando o seguinte comando:

${nome_da_variavel}

Oque deicaria o comando da seguinte mandeira:

alert("A soma dos números é igual a: ${n3}")

Porém a utilidade desse método fica mais clara quando utilizamos mais de uma variavel, veja a seguir:

alert("A soma dos números" +n1+ " + " +n2+ " é igual a: " +n3)

Seria transformado em:

alert("A soma dos números ${n1} + ${n2} é igual a: ${n3}")

Comandos com variáveis String!

Existem alguns comandos interessantes a serem aprendidos, como por exemplo:

document.writeln("Olá mundo!")

Que serve para enviar um texto do javaScript para o seu documento html.

nome_da_var.toUpperCase

Que serve para deixar todas as letras de uma string em caixa alta(maiúsculas).

nome_da_var.toLowerCase

Para deixar todas as letras em minúsculo.

nome_da_var.length

Que serve para mostrar quantas letras possui uma string.

Comandos com variáveis Number

Existem alguns comandos interessantes a serem aprendidos, como por exemplo:

nome_da_var.toFixed(2)

Que serve para que sejam mostrados duas casas decimais, e não apenas uma!

nome_da_var.replace('.' , ',')

Que serve para que a separação das casas decimais seja feita por uma vírgula ou qualquer caractere de sua preferencia!

nome_da_var.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})

Que serve para que o number seja exibido como uma moeda local, que no caso do nosso exemplo é o real!
