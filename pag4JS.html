Operadores Aritméticos

Estes são os simbolos utilizados para cálculos no JavaScript, sendo eles:

+

Para soma.

-

Para subtração.

*

Para multiplicação.

/

Para divisão.

%

Para descobrir o resto de uma divisão de números inteiros.

**

Para potência.

Operadores de atribuição

São utilizados para atribuír valor a variaveis, como por exemplo:

var a = 5 + 3

Assim nos atribuímos para a variavel "a" o valor da soma.

var a = a + 3

Dessa forma declaramos que "a" recebe seu valor anterior mais 3.

a+=3

Estes comando funciona exatamente como o anterior, porem de frma simplificada. Esse método só pode ser utilizados em situações identicas ao exemplo anterior, podendo varia apenas entre os sinais! (+, - ,* , **, /, %)

var b = a

Dessa forma declaramos que a variável "b" terá o mesmo valor que "a".
