⬅ Início

Lista ordenada

Para utilizarmos a lista ordelana iremos por a tag <ol>, e para cada item da lista utilizamos a tag <li>.

Oque resultaria no código a seguir:

    <ol>
        <li>Arcordar</li>
        <li>Tomar café</li>
        <li>Tomar banho</li>
        <li>Escovar os dentes</li>
    </ol>

Porém existem alguns tipos de listas, Temos como opções de type: 1 = lista em ordem numérica, A = lista em ordem alfabetica com letras maiúsculas e "a" para minúsculas, I = Ordem em números romanos tendo "i" para deixar minúsculo

Para utilizarmos esses tipos basta utilizamos o seguinte comando ao lado da tag <ol>:

<ol type="1"></ol>

Com ele nos criamos uma lista numérica, deixando o código da seguinte maneira:

    <ol type="1"></ol>
        <li>Arcordar</li>
        <li>Tomar café</li>
        <li>Tomar banho</li>
        <li>Escovar os dentes</li>
    </ol>

Que resultaria na seguinte lista:

  1. Arcordar
  2. Tomar café
  3. Tomar banho
  4. Escovar os dentes
