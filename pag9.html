Criando tabelas em HTML

Tabelas podem ser importante para demostrar comparações de dados, como por exemplo a tabela a seguir:

Para criarmos uma tabela em html utilizamos o comando é recomendado criar uma tag table, apos isso criamos as linhas com a tag tr, nela criamos as células com a tag td!

oque deixaria o código assim:

<table>

<tr>

<td>A1</td>

<td>B1</td>

<td>c1</td>

</tr>

<tr>

<td>A2</td>

<td>B2</td>

<td>c2</td>

</tr>

<tr>

<td>A3</td>

<td>B3</td>

<td>c3</td>

</tr>

Oque resultaria nisso:

A1 B1 c1
A2 B2 c2
A3 B3 c3
